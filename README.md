# SIP Statistical Analysis Pipeline

Automated pipeline for Substance-Induced Psychosis (SIP) statistical analysis.

**Live Dashboard**: [https://ntwkkm.github.io/bossissa/](https://ntwkkm.github.io/bossissa/)

## Overview

- **Data Source**: [Google Sheets](https://docs.google.com/spreadsheets/d/1JG5cGNaqo_2DB4Z7N-3w8Yz8eHkQhAhDLoUxMoymlAY/edit?usp=sharing) (auto-fetched via public CSV export)
- **Analysis**: Python OOP TableOne generator + LASSO-driven multivariate logistic regression (Firth + Standard MLE)
- **Trigger**: Manual GitHub Actions (`workflow_dispatch`)
- **Output**: Interactive GitHub Pages dashboard with 4 tabs

## Dashboard Tabs

| Tab | Content |
|-----|---------|
| 📊 **Table One** | Baseline characteristics stratified by SIP diagnosis, with filters, search, CSV export |
| 📈 **Infographics** | 8 interactive Plotly charts (donut, symptoms, substances, age, DSM-5, UDS, discharge, forest plot) |
| 📋 **Frequency Stats** | Distribution tables stratified by inclusion criteria, psychiatric history, and SIP (all groups) |
| 🧬 **Multivariate** | LASSO feature selection → Firth + Standard logistic regression side-by-side with interpretation |

## Stratification

All variables are compared between:
- **SIP** (`สรุป: เข้าเกณฑ์ SIP = 1`): Meets DSM-5 SIP criteria
- **Not SIP** (`สรุป: เข้าเกณฑ์ SIP = 0`): Does not meet criteria
- Rows with `= 99` (incomplete data) are excluded from primary analysis (retained in Frequency Stats tab).

## Running the Analysis

1. Go to **Actions** tab in this repository
2. Select **SIP Statistical Analysis**
3. Click **Run workflow** → **Run workflow** (green button)
4. Wait ~3 minutes for the job to complete
5. GitHub Pages will auto-update

## Statistical Methods

### Univariate (Table One)

| Variable Type | Test | Reported As |
|---|---|---|
| Continuous Normal | Welch t-test | Mean ± SD |
| Continuous Non-Normal | Mann-Whitney U | Median [Q1, Q3] |
| Categorical | Chi-square / Fisher's Exact | n (%) |

Effect sizes: SMD (Standardized Mean Difference), OR with 95% CI.

### Multivariate

1. **LASSO** (L1-regularized logistic regression) — feature selection by AIC across C=0.1, 0.5, 1.0
2. **Firth Logistic Regression** (`firthmodels`) — penalized likelihood with profile likelihood CIs
3. **Standard MLE** (`statsmodels`) — Wald CIs, pseudo R² for model fit

Both models run on LASSO-selected features and are presented side-by-side for comparison.
