Statistical Methods

📊

Variable Classification

Automatic type inference using Shapiro-Wilk (N<5,000) or Jarque-Bera (N≥5,000) normality tests, with descriptive criteria (|skewness|<1.0 and |kurtosis|<2.0 for N>50).

🔬

Continuous Variables

Welch t-test (normal, 2 groups) or Mann-Whitney U (non-normal, 2 groups). Reported as mean ± SD or median [Q1, Q3].

📋

Categorical Variables

Pearson Chi-square with fallback to Fisher's Exact test for 2×2 tables when minimum expected frequency <5. Reported as n (%).

⚖️

Effect Size

Standardized Mean Difference (SMD) with pooled SD. Odds Ratio (OR) with 95% CI using Haldane-Anscombe correction for categorical and univariate logistic regression for continuous variables.
