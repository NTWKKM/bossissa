# SIP Statistical Analysis Pipeline

Automated pipeline for Substance-Induced Psychosis (SIP) statistical analysis.

**Live Dashboard**: [https://ntwkkm.github.io/bossissa/](https://ntwkkm.github.io/bossissa/)

## Overview

- **Data Source**: [Google Sheets](https://docs.google.com/spreadsheets/d/1JG5cGNaqo_2DB4Z7N-3w8Yz8eHkQhAhDLoUxMoymlAY/edit?usp=sharing) (auto-fetched via public CSV export)
- **Analysis**: Python OOP TableOne generator + LASSO-driven multivariate logistic regression (Firth + Standard MLE)
- **Trigger**: Manual GitHub Actions (`workflow_dispatch`)
- **Output**: Interactive GitHub Pages dashboard with 4 tabs

## Statistical Methods

### Univariate (Table One)

| Variable Type | Test | Reported As |
|---|---|---|
| Continuous Normal | Welch t-test | Mean В± SD |
| Continuous Non-Normal | Mann-Whitney U | Median [Q1, Q3] |
| Categorical | Chi-square / Fisher's Exact | n (%) |

Effect sizes: SMD (Standardized Mean Difference), OR with 95% CI.

### Multivariate

1. **LASSO** (L1-regularized logistic regression) вЂ” feature selection by AIC across C=0.1, 0.5, 1.0
2. **Firth Logistic Regression** (`firthmodels`) вЂ” penalized likelihood with profile likelihood CIs
3. **Standard MLE** (`statsmodels`) вЂ” Wald CIs, pseudo RВІ for model fit

Both models run on all features (post-collinearity removal) and are presented side-by-side for comparison.
