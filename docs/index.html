Statistical Methods

📊

Variable Classification

Automatic type inference using Shapiro-Wilk (N<5,000) or Jarque-Bera (N≥5,000) normality tests, with descriptive criteria (|skew|<1.0 and |kurt|<2.0 for N>50).

🔬

Continuous Variables

Welch t-test (normal, 2 groups) or Mann-Whitney U (non-normal). Reported as mean ± SD or median [Q1, Q3].

📋

Categorical Variables

Pearson Chi-square with fallback to Fisher's Exact test for 2×2 tables when min expected frequency <5. Reported as n (%).

⚖️

Effect Size

SMD with pooled SD. OR with 95% CI via Haldane-Anscombe correction (categorical) or univariate logistic regression (continuous).
