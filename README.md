# SIP Statistical Analysis Pipeline

Automated pipeline for Substance-Induced Psychosis (SIP) statistical analysis.

**Live Dashboard**: [https://ntwkkm.github.io/bossissa/](https://ntwkkm.github.io/bossissa/)

## Overview

- **Data Source**: [Google Sheets](https://docs.google.com/spreadsheets/d/1JG5cGNaqo_2DB4Z7N-3w8Yz8eHkQhAhDLoUxMoymlAY/edit?usp=sharing) (auto-fetched via public CSV export)
- **Analysis**: Python OOP TableOne generator with full statistical engine
- **Trigger**: Manual GitHub Actions (`workflow_dispatch`)
- **Output**: Interactive GitHub Pages dashboard

## Stratification

All variables are compared between:
- **SIP** (`สรุป: เข้าเกณฑ์ SIP = 1`): Meets DSM-5 SIP criteria
- **Not SIP** (`สรุป: เข้าเกณฑ์ SIP = 0`): Does not meet criteria
- Rows with `= 99` (incomplete data) are excluded.

## Running the Analysis

1. Go to **Actions** tab in this repository
2. Select **SIP Statistical Analysis**
3. Click **Run workflow** → **Run workflow** (green button)
4. Wait ~2 minutes for the job to complete
5. GitHub Pages will auto-update

## Statistical Methods

| Variable Type | Test | Reported As |
|---|---|---|
| Continuous Normal | Welch t-test | Mean ± SD |
| Continuous Non-Normal | Mann-Whitney U | Median [Q1, Q3] |
| Categorical | Chi-square / Fisher's Exact | n (%) |

Additional: SMD (Standardized Mean Difference), OR with 95% CI.

## Prerequisites (Google Sheet)

The Google Sheet must be set to **"Anyone with the link can view"** (public).

> ⚠️ Ensure no PHI is present in the sheet before making it public.
